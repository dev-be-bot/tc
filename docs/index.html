Powerful Features

Everything You Need to Run Support

Private threads, automation rules, smart assignment, and real-time analytics — all out of the box.

Private Thread Tickets

Every message in your support channel creates a private thread. Only the user and staff can see it — no channel clutter.

Web Dashboard

Ticket volume charts, resolution tracking, staff metrics, priority breakdowns, and satisfaction scores — all in one place.

Automation Rules

Keyword and regex rules that auto-assign staff, set priority, delete messages, or send custom DMs with embeds.

Priority Escalation

Tickets auto-escalate from New → Medium → Escalating → Urgent based on your configurable time thresholds.

Smart Assignment

All Staff, Round Robin, or Primary Owner strategies. Override per-rule for specific keywords or categories.

Satisfaction Ratings

Users receive a star rating DM after ticket closure. Track ratings per staff member and overall trends.
