Simple Setup

Get ThreadCord running on your server in minutes

1. Create Channels

Set up two channels: one for ticket creation (e.g., #support-tickets) and one for logs (e.g., #ticket-logs).

2. Define Staff Role

Select or create a role that will have permission to answer tickets (e.g., @Support, @Moderator).

3. Run /setup

Use /setup channel:#support-tickets staff_role:@Support logs_channel:#ticket-logs

How Tickets Work

1

User Creates Ticket

A member sends any message in the ticket channel. ThreadCord automatically creates a private thread with ticket controls and logs it to your dashboard.

2

Staff Manages Ticket

Staff members can use intuitive buttons to mark tickets as done, assign them, or close them. All actions are tracked in real-time.

3

Track Everything

All ticket activities are logged to your designated channel and displayed in the comprehensive web dashboard with analytics, ratings, and staff performance metrics.
