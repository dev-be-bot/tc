## How to Use

1. Invite ThreadCord to your server using the **Invite Bot** badge above.  
2. Create a channel for support tickets.  
3. Use `/setup` to configure the bot (channels, roles, permissions).  
4. Your community can now create tickets by simply replying to the specified channel.
